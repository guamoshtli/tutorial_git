Para revertir un cambio en el Working directory
git restore file o bien ..- git checkout -- file
Para revertir un cambio en el staged git restore --staged file..." to unstage o bien ... git reset HEAD file
Para revertir un cambio en el stage con commit realizado
git checkout [hash commit] o bien ... git reset --hard [hash commit]
Para revertir un cambio en el stage con commit realizado al stage sin comit:
git reset --soft [hash commit] al working ... git restore --staged file del working .. git restore file
